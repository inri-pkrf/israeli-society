# Israeli Society Learning Platform

An interactive educational web application developed with React.

## About the Project

This project is an educational learning platform designed to provide users with an engaging and interactive learning experience. The application presents educational content through a modern and user-friendly interface, encouraging active participation and knowledge development.

## Features

* Interactive learning experience
* Modern and responsive user interface
* Educational activities and exercises
* Progress tracking
* User-friendly navigation
* Mobile and desktop compatibility

## Technologies Used

* React
* JavaScript
* HTML5
* CSS3

## Getting Started

### Installation

```bash
npm install
```

### Run the Application

```bash
npm start
```

The application will run in development mode and can be accessed through:

```text
http://localhost:3000
```

### Production Build

```bash
npm run build
```

This command creates an optimized production build in the `build` folder.

## Project Purpose

The goal of this project is to provide an accessible and interactive educational environment that supports learning through digital tools and engaging activities.
